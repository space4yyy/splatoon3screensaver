# 斯普拉遁 3 屏幕保护程序 (Splatoon 3 Screensaver)

这是一个为 macOS 原生编写的屏幕保护程序 (`.saver`)，完美重现了 Shadertoy 上的经典着色器 `f3SGWc`（由 Nick27 制作的 《斯普拉遁 3 启动动画》）。

本程序为非官方同人创作，与任天堂（Nintendo）没有任何关联或代言关系。

---

## 功能特点

* **原生 Metal 与 CAMetalLayer 架构**：直接基于 `CAMetalLayer` 渲染，采用扁平化无子视图设计，完美解决 macOS 系统下外接显示器、多屏幕及多 GPU 渲染黑屏的问题，运行极速稳定。
* **高帧率与显示器同步 (VSync)**：支持 30 FPS、60 FPS、120 FPS 限制。当选择“显示器同步”时，渲染循环将自动与您的显示器物理刷新率（如 100Hz、120Hz、144Hz 等）完美垂直同步，无画面撕裂与多余 CPU 开销。
* **智能自适应屏幕比例**：针对 21:9 带鱼屏、16:9、16:10 以及竖屏等各种异形显示器进行了动态视口和矩阵修正，确保水泡与墨水旋转等元素始终保持完美的正圆形，绝不拉伸变形。
* **历代斯普拉遁游戏色彩方案**：
  * **历代随机**：每次启动屏保时，随机选择一代的配色。
  * **历代循环**：随时间自动在斯普拉遁 1、2、3 代的经典配色之间平滑渐变过渡。
  * **斯普拉遁 1 代**：经典 橙色 / 蓝色。
  * **斯普拉遁 2 代**：经典 粉色 / 绿色。
  * **斯普拉遁 3 代**：经典 黄色 / 紫色。
  * **自定义**：允许在选项面板中自由选择冷暖两色墨水。
* **渲染缩放调整**：可调整内部仿真分辨率（0.5x 到 1.5x），在低配设备或超高分辨率（如 5K/6K）屏幕下优化性能。
* **静音设计**：应用户要求，去除了启动音效，适合安静的办公或居家环境。
* **自定义预览图**：已配置斯普拉遁 3 主题霓虹墨水高清预览图，在系统设置中完美显示。

## 构建步骤

在终端中运行以下命令：

1. **编译屏幕保护程序包**：
   ```sh
   make
   ```
   编译好的安装包将输出至：`build/Splatoon3Screensaver.saver`

2. **安装到当前用户系统**：
   ```sh
   make install
   ```
   程序将自动安装至 `~/Library/Screen Savers/` 目录，并强制重载系统屏保服务。

3. **配置使用**：
   打开 **系统设置 -> 屏幕保护程序**，找到 `Splatoon3Screensaver`（或 `Splatoon 3 Boot`），点击 **选项** 即可配置色彩方案、帧率和缩放比例。

## 分发与给他人安装使用

如果您想把这个屏保分享给其他人，请按以下步骤操作：

1. **打包发送**：
   * 在 Finder 中找到编译好的 `build/Splatoon3Screensaver.saver`。
   * 右键点击它，选择 **压缩 "Splatoon3Screensaver.saver"**，生成 `Splatoon3Screensaver.saver.zip`。
   * 将该 `.zip` 压缩包发送给其他用户。

2. **其他用户的安装与白名单解除（关键步骤）**：
   由于该屏幕保护程序是本地 ad-hoc 签名且未向 Apple 注册开发者证书，macOS 安全机制（Gatekeeper）会默认拦截它，可能导致“无法打开”或运行后“外接显示器黑屏”。请让对方按照以下步骤操作：
   
   * **解压并复制**：解压 `.zip` 压缩包，将 `Splatoon3Screensaver.saver` 拖入用户的屏保目录下：
     `~/Library/Screen Savers/`
     *(小贴士：在 Finder 中按下 `Cmd + Shift + G`，粘贴路径 `~/Library/Screen Savers/`，回车即可直达)*
   * **解除 Quarantine 隔离标记**：打开 **终端 (Terminal)** 应用程序，复制并执行以下命令：
     ```sh
     xattr -cr ~/Library/Screen\ Savers/Splatoon3Screensaver.saver
     ```
   * **启用屏保**：进入 **系统设置 -> 屏幕保护程序**，启用即可正常工作！

## 自定义预览图说明

屏幕保护程序在系统设置中的预览缩略图是可以完全自定义的：
* macOS 系统通过读取屏保包内 `Resources/` 目录下的 `thumbnail.png` 和 `thumbnail@2x.png` 来显示预览图。
* **标准规格**：
  * `thumbnail.png`: 90 x 58 像素
  * `thumbnail@2x.png`: 180 x 116 像素
* **当前配置**：我们已经使用 AI 自动生成了斯普拉遁 3 经典黄紫墨水飞溅的视觉图，并使用系统图像处理工具 `sips` 进行了中心裁剪和分辨率缩放，内置在屏保包中。如果您日后想更换它，只需替换项目 `Resources/` 目录下的这两个 PNG 文件并重新运行 `make install` 即可。

## 调试与着色器开发

如果您需要修改或开发着色器，我们内置了专门的调试日志。日志实时输出在：
`/tmp/Splatoon3Screensaver.log`

您可以通过终端命令实时查看渲染状态、各显示器的分辨率、帧渲染耗时等：
```sh
tail -f /tmp/Splatoon3Screensaver.log
```
